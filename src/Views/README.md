**Pull Repo :**

Sebelum Coding Bisa Di Pull dahulu setiap mau ngoding agar tidak terjadi conflict
atau via commadline :

```
    git pull origin main
```

**Branch :**

buat branch baru biar bisa kode masuk ke github ku kalau langsung ke main ku dah atur ku tolak jadi inget buat branch bisa di github desktop atau di commandline :
```
    git checkout -b <nama-branch>
```
Contoh :
```
    git checkout branch -b model-wira
```

**Add and Push** : 
setelah codingan selesai di github dekstop bisa melakukan add dahulu setelah itu commit dan langsung push pastikan di branch yang telah dibuat pushnya cara di commandline :
```
    git add <directory / file / * (semua file)->
```
```
    git push origin <nama-branch>
```
